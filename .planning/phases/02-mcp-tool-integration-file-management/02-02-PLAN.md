**Required tests (8-12 tests):**

1. **test_tool_returns_list_on_valid_input** — Call `generate_source_report(VALID_CITATION_JSON)`, assert result is a list with length 2.

2. **test_tool_returns_text_content_with_download_link** — Verify result[0] is TextContent with type="text" and text containing "download" (case-insensitive).

3. **test_tool_returns_embedded_resource_with_pdf** — Verify result[1] is EmbeddedResource with resource.mimeType == "application/pdf" and resource.blob is a non-empty string (base64 encoded).

4. **test_tool_validates_base64_encoding** — Verify result[1].resource.blob can be decoded with `base64.b64decode()` without error, and the decoded bytes start with `%PDF` (PDF magic bytes).

5. **test_tool_saves_pdf_to_data_dir** — After calling the tool, verify a .pdf file exists in DATA_DIR with the expected filename pattern. Clean up the file after test.

6. **test_tool_filename_includes_model_name_and_sources** — Call with model_name="MyTestModel", verify the saved filename starts with "MyTestModel_sources_" and ends with ".pdf". Clean up after test.

7. **test_tool_filename_default_model_name** — Call without model_name parameter, verify filename starts with "model_sources_". Clean up after test.

8. **test_tool_sanitizes_model_name** — Call with model_name="Bad/Name\\With..Chars!", verify filename only contains safe characters (alphanumeric, underscore, hyphen). Clean up after test.

9. **test_tool_returns_list_on_invalid_json** — Call with "not valid json", verify result is a list with length 1 containing TextContent with "Validation Error" in text.

10. **test_tool_returns_list_on_schema_error** — Call with `json.dumps({"citations": []})` (empty citations list), verify result is list with TextContent containing error message.

11. **test_cleanup_handles_pdf_extension** — Import cleanup_old_files source via inspect, verify the source code contains `".pdf"` in the endswith check. (Do NOT actually run the cleanup thread in tests.)

12. **test_pdf_content_type_detection** — Use `mimetypes.guess_type("test.pdf")` to verify Python's mimetypes module returns "application/pdf", confirming FileResponse will serve PDFs correctly.
