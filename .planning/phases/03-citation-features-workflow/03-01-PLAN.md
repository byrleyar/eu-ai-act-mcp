**1. Add CONFIDENCE_COLORS constant** (module level, after font registration):
```python
CONFIDENCE_COLORS = {
    'DIRECT': colors.HexColor('#D4EDDA'),      # Light green
    'INFERRED': colors.HexColor('#FFF3CD'),    # Light yellow
    'DEFAULT': colors.HexColor('#E7E8EA'),     # Light gray
    'NOT FOUND': colors.HexColor('#F8D7DA')    # Light red
}
```
These are WCAG AA compliant colors (4.5:1+ contrast with black text).

**2. Add executive summary section** as a helper function `_build_executive_summary(citations, model_card_id)`:
- Use `collections.Counter` to tally confidence levels
- Return list of flowables: header Paragraph ("Executive Summary"), Spacer, body Paragraph with model_card_id, total questions, confidence breakdown (count + percentage for each of DIRECT/INFERRED/DEFAULT/NOT FOUND)
- Handle zero citations gracefully (show "No citations provided")
- Use `<b>` HTML tags in Paragraph text for bold labels, `<br/>` for line breaks
- Use `datetime.now().strftime('%Y-%m-%d %H:%M')` for timestamp in summary

**3. Enhance table content per confidence level** (CITE-01 through CITE-05):
Keep the same 7-column table structure but format cell content differently based on confidence:
- Column 1 (#): Show question_id instead of row number (CITE-08 cross-referencing)
- Column 2 (Question): question_text (unchanged)
- Column 3 (Answer): answer (unchanged)
- Column 4 (Source/Details): Content varies by confidence level:
  - DIRECT: Source quote wrapped in `<i>` tags (italics) with section attribution: `"<i>{quote}</i><br/><br/>Section: {section}"`
  - INFERRED: Related quote(s) + reasoning chain: `"Related: <i>{quote}</i><br/><br/>Reasoning: {reasoning}"`
  - DEFAULT: Reasoning explains why standard value used: `"Standard value applied.<br/><br/>Rationale: {reasoning}"`
  - NOT FOUND: Documents what was searched for: `"Information not found.<br/><br/>Searched: {reasoning}"`
- Column 5 (Section): source_section (keep for DIRECT/INFERRED, show dash for DEFAULT/NOT FOUND)
- Column 6 (Confidence): confidence level text label (unchanged -- provides text label alongside color)
- Column 7 (Reasoning): reasoning text (unchanged)

**Important: Only DejaVuSans.ttf is available** (no Bold or Italic font variants). ReportLab's `<i>` and `<b>` tags in Paragraph markup will simulate italic/bold using the base font with glyph transformations. This is acceptable for visual hierarchy.

**4. Apply confidence-based row colors** via TableStyle:
After building the base TableStyle commands, iterate over citations and append per-row BACKGROUND commands using CONFIDENCE_COLORS. Remove the alternating row backgrounds (`ROWBACKGROUNDS` command) as confidence colors replace them.

**5. Update story construction order**:
1. Title ("AI Act Compliance - Source Citation Report")
2. Subtitle with timestamp
3. Spacer
4. Executive summary flowables (from `_build_executive_summary`)
5. Spacer
6. Table with confidence-colored rows

**6. Increase bottomMargin** from 0.5 * inch to 0.75 * inch to leave room for footer (Task 2 adds footer callback).

**XML escaping**: Continue using `_escape_xml()` for all user text. Apply escaping BEFORE wrapping in HTML tags like `<i>`. For example: `f"<i>{_escape_xml(quote)}</i>"`.

Do NOT change the module docstring exports or the function's position in the file.
