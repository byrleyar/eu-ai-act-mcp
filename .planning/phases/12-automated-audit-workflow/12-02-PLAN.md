<task type="auto">
  <name>Task 1: Create audit_processor.py CLI and unit tests</name>
  <files>audit_processor.py, tests/test_audit_processor.py</files>
  <action>
Create audit_processor.py following the batch_processor.py pattern exactly:

1. **Imports**: typer, rich (Console, Progress, SpinnerColumn, TextColumn, BarColumn, TaskProgressColumn), pathlib.Path, csv, json, time, datetime. From audit_service import AuditService, AuditResult. From compliance_service import ProcessingContext.

2. **app = typer.Typer(help="EU AI Act Compliance Audit Processor")**

3. **@app.command() def run(batch_dir, model_filter)**:
   - `batch_dir: Path = typer.Argument(..., help="Path to batch directory to audit")`
   - `model_filter: Optional[str] = typer.Option(None, "--model", "-m", help="Audit single model only (directory name)")`
   - Validate batch_dir exists, exit with error if not
   - Discover model subdirectories: iterate batch_dir children, include only directories that contain model_card.txt
   - If model_filter provided, filter to matching directory only
   - If no model dirs found, print warning and exit
   - Initialize AuditService() (uses ANTHROPIC_API_KEY from env)
   - For each model dir (with rich Progress bar, same style as batch_processor.py):
     a. Extract model_id: reverse the sanitize_model_id pattern (replace "__" with "/")
     b. Create ProcessingContext with model_id=model_id, output_path=str(model_dir), data_dir=str(model_dir)
     c. Call service.audit_model(ctx) in try/except
     d. On success: log score summary (e.g., "Scores: 1=X, 1i=Y, 2=Z, 3=W, 4=V")
     e. On failure: log error, record in results, continue to next model
   - After loop: generate audit_summary.csv in batch_dir with columns: model_id, status, duration, total_fields, score_1, score_1i, score_2, score_3, score_4, error
   - Print summary table to console

4. **if __name__ == "__main__": app()**

Create tests/test_audit_processor.py:
- test_model_dir_discovery: Create a temp dir with 2 subdirectories (one with model_card.txt, one without). Assert only the one with model_card.txt is discovered.
- test_model_id_from_dirname: Assert "facebook__opt-125m" reverses to "facebook/opt-125m".
- test_audit_summary_csv_format: Assert the CSV writer produces correct headers.
  </action>
