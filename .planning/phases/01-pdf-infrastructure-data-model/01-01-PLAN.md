<task type="auto">
  <name>Task 1: Create citation schema with Pydantic validation</name>
  <files>citation_schema.py</files>
  <action>
Create `citation_schema.py` at project root (same level as `docx_generator.py` and `server.py`).

Define the following:

1. **ConfidenceLevel enum** (str, Enum) with four values:
   - DIRECT = "DIRECT"
   - INFERRED = "INFERRED"
   - DEFAULT = "DEFAULT"
   - NOT_FOUND = "NOT FOUND"

2. **Citation model** (Pydantic BaseModel) with fields:
   - `question_id: str` - Field(min_length=1), unique question identifier
   - `question_text: str` - Field(min_length=1), full question text
   - `answer: str` - Field(default=""), answer text (may be empty string for NOT FOUND)
   - `source_quote: str` - Field(default=""), relevant quote from source document
   - `source_section: str` - Field(default=""), section identifier in source
   - `confidence: ConfidenceLevel` - confidence level for this citation
   - `reasoning: str` - Field(min_length=1), explanation of confidence level assignment

   Add a `@field_validator` for `question_text` and `reasoning` that rejects whitespace-only strings (strip and check empty). Use Pydantic v2 syntax: `@field_validator('field', mode='after')` with `@classmethod`.

3. **CitationReport model** (Pydantic BaseModel) with:
   - `citations: list[Citation]` - Field(min_length=1), at least one citation required

4. **validate_citation_json function** that:
   - Accepts a JSON string
   - Calls `CitationReport.model_validate_json(json_string)`
   - On success, returns the CitationReport instance
   - On ValidationError, formats errors using `e.errors()` to produce field-level messages like:
     `"citations -> 0 -> confidence: Input should be 'DIRECT', 'INFERRED', 'DEFAULT' or 'NOT FOUND' (type: enum)"`
   - Raises ValueError with the formatted error string

Import only from: `pydantic` (BaseModel, Field, ValidationError, field_validator), `enum` (Enum). Both are already in requirements.txt.

Do NOT import or reference ReportLab or PDF generation -- this module is purely data validation.
